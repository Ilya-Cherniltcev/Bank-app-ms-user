### Микросервис банковского приложения MsUser ###

Командный проект: Ilya Cherniltcev, Dmitrii Arzhanov
***
### Функционал: ###

Написана backend-часть REST микросервиса, реализующая CRUD операции и позволяющая работать с информацией о пользователях, а именно:
- создавать, редактировать (поля "name", "passport", "phoneNumber"), удалять пользователя;
- получать всех пользователей, либо одного по id, номеру телефона или паспорта;
- получать пользователей по дате регистрации;
- получать пользователей по части номера паспорта или телефона.

***
### Стек, используемых технологий: ###

Язык программирования - **Java**

База данных - **MySQL**



Фреймворки/Библиотеки: **Spring Boot**, **JPA**, **Lombok**, **Liquibase**, **Mapstruct**, **SpringDoc OpenAPI UI**, **JUnit**, **Mockito**, 

ПО для контейнеризации - **Docker**
